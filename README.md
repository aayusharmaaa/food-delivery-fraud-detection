# Fraud Detection MVP

## Setup

1.  **Backend**:
    *   Navigate to `backend/`
    *   Create venv: `python -m venv venv`
    *   Activate: `venv\Scripts\activate`
    *   Install: `pip install -r requirements.txt`
    *   Run: `uvicorn main:app --reload`

2.  **Frontend**:
    *   Navigate to `frontend/`
    *   Install: `npm install`
    *   Run: `npm run dev`

## Quick Start (Windows)

Double-click `start_servers.bat` to launch both backend and frontend.

## Usage

1.  Open the frontend URL (usually `http://localhost:5173`).
2.  Upload an image.
3.  View the Risk Score and Analysis.
